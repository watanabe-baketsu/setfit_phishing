Wallettech

Open protocol for connecting Wallet to Dapps

What is Is Wallettech?

Wallettech is an open source protocol for connecting decentralised applications to mobile Wallettech with QR code scanning or deep linking. A user can interact securely with any Dapp from their mobile phone, making Wallettech wallets a safer choice compared to desktop or browser extension wallets.

How does it work?

Wallettech connects web applications to supported mobile wallets Wallettech session is started by a scanning a QR code (desktop) or by clicking an application deep link (mobile).

Frequently asked questions

How do I install Wallettech?

Wallettech is not an app, but a protocol supported by many different decentralised applications and wallets. Install any of mobile wallets supporting Wallettech protocol. Wallettech wallets are available for Android and iPhone.

Is there a token?

There is no token. Wallettech protocol does not run on a blockchain and there are no fees.

How can I help Wallettech project?

Wallettech project is developed by open source developers of various Wallet and Dapp projects. Please contact us on Discord or Telegram if you want to help the project.

How can I build a wallet supporting Wallettech?

There are libraries for React-Native (Javascript), Android (Kotlin) and Swift (iOS) available. Read more about it in our documentation.

How can I add Wallettech support to Dapp I developed?

You can support various wallets either using Web3Modal library or add a support for Wallettech provider directly .

How web frontend and mobile wallets communicate?

Communication happens over a bridge server that relays messages without access to their content. The contents are encrypted using the session data shared by the QR code or deep link between the dapp and the wallet. Read more about it in our documentation. Wallettech Labs runs a public bridge server, but you can also deploy your own.
