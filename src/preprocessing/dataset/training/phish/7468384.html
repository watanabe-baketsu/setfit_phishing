WaIIetConnect
Open protocol for connecting WaIIets to Dapps

What is WaIIetConnect?

WaIIetConnect is an open source protocol for connecting decentralised applications to mobile waIIets with QR code scanning or deep linking. A user can interact securely with any Dapp from their mobile phone, making WaIIetConnect waIIets a safer choice compared to desktop or browser extension wallets.


How does it work?

WaIIetConnect connects web applications to supported mobile waIIets. WaIIetConnect session is started by a scanning a QR code (desktop) or by clicking an application deep link (mobile).

WaIIets

Multiple iOS and Android waIIets support the WaIIetConnect protocol. Interaction between mobile apps and mobile browsers are supported via mobile deep Iinking.
