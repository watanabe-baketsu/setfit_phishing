What is Block2Node?

Block2Node is an open source protocol for connecting decentralised applications to mobile wallets with QR code scanning or deep linking. A user can interact securely with any Dapp from their mobile phone, making Block2 wallets a safer choice compared to desktop or browser extension wallets.

How Does it Work?

Block2Node connects web applications to supported mobile wallets. Block2Node session is started by a scanning a QR code (desktop) or by clicking an application deep link (mobile).
